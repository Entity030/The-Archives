    What's New THE ARCHIVES

    Changelog

    New v4.0 — Polish & Analytics Jun 2025
    • Analytics Dashboard — view total script views, most popular scripts, recently viewed, and favourites (visible after admin unlock)
    • Per-script view counts — each card now shows how many times a script has been opened, and the detail modal shows the count in metadata
    • Accent colour picker — choose from preset hues or any custom colour; persists across sessions
    • High-contrast mode — toggle via the AA button in the header for WCAG-compliant contrast
    • Particle bursts — favouriting a script or copying code triggers a satisfying particle animation
    • Swipe-to-close — on mobile, swipe the bottom sheet downward to dismiss it
    • Better empty states — each filter has its own icon and message with a Clear Filters action button
    • Accessibility — skip-to-content link, improved ARIA labels and roles, screen-reader labels on code blocks, reduced-motion support
    New v3.0 — Discovery Jun 2025
    • Advanced Search — full-text + date range filter, multi-select tag chips, and boolean toggles for DataStore, Mobile-friendly, and Game Showcase
    • Saved Searches — bookmark any combination of query + filters as a one-click chip
    • For You — personalized recommendations based on view history and favorites, with per-script reason labels
    • Collections / Script Packs — create named bundles with emoji icons, add scripts from the detail modal, and share via deep-link URL
    • Beginner Path: progress bar, "Done / Next" badges, path-complete celebration banner, and manual reset
    New v2.0 — Community Jun 2025
    • Script ratings — rate 1–5 stars and tag scripts as Useful, Buggy, or Too Complex
    • "Community Pick" badge — automatically awarded to scripts with ≥4.5 stars and 3+ ratings
    • Discussion threads — post comments and questions directly inside each script's modal
    • Moderated Queue — admins can review, approve, or reject community submissions from a new tab in the Admin panel
    • Queue supports JSON paste imports alongside live Google Form submissions
    New v1.4 — Feature Drop Jun 2025
    • Search now matches inside script code — toggle the "Search Code" button in the toolbar
    • Print-friendly script view — clean white layout for printing or saving as PDF
    • Roblox Studio Template button — opens a pre-configured template place with the script
    • Dark & light syntax highlighting — code theme now follows your UI theme automatically
    • Changelog modal — this panel, so you always know what's changed
    • Empty state now shows an illustrated Roblox-style character when no scripts are found
    • Theme now respects your system's prefers-color-scheme on first visit
    New v1.3 — Polish & UX May 2025
    • URL deep-linking — share or bookmark a direct link to any script
    • Mobile bottom sheet — smooth slide-up modal on small screens
    • Keyboard navigation — arrow keys to browse cards, / to focus search
    • Script editing — admins can edit any existing script in-place
    • Export & import backup — download or restore the full script library as JSON
    • View count tracking — each script tracks how many times it has been opened
    • Syntax highlighting via Highlight.js — full Luau colour support
    Launched v1.0 — Initial Release Apr 2025
    • Core library with 7 production-ready Luau scripts
    • Live search, filter buttons (Side, Difficulty, Favorites, Beginner Path)
    • Admin panel with password gating, add & delete scripts
    • Dark / light theme toggle with localStorage persistence
    • Favorites / bookmarks system
    • Beginner Path — curated 3-step learning sequence
    • Related Scripts recommendations in the detail modal
    • "Made With This Script" Roblox game showcase
